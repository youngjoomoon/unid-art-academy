# 유니드아트 미술학원 홍보 웹사이트

아동·초등 미술 + 성인 취미 미술을 위한 단일 페이지 홍보 사이트.
순수 HTML/CSS/JS로 만들어져 별도 설치·빌드 없이 동작합니다.

## 보는 법
`index.html`을 더블클릭하면 브라우저에서 바로 열립니다.

## 구조
```
미술학원/
├── index.html      # 페이지 전체 (섹션: 히어로, 소개, 수업, 갤러리, FAQ, 상담)
├── css/style.css   # 디자인 (파스텔 톤, 반응형, 애니메이션)
└── js/main.js      # 모바일 메뉴, 스크롤 효과
```

## 나중에 채워야 할 것 (index.html에서 `TODO` 검색)
| 항목 | 위치 | 방법 |
|---|---|---|
| 전화번호 | 상담 문의 섹션 | `000-0000-0000` 두 곳을 실제 번호로 교체 |
| 주소 | 오시는 길 카드 | "주소 준비 중" 교체 + 네이버/카카오맵 iframe 추가 |
| 카카오톡 채널 | 카카오톡 상담 카드 | `href="#"`를 채널 URL로 교체, `onclick` 삭제 |
| 작품 사진 | 갤러리 섹션 | `.gallery__item` 안의 이모지 placeholder를 `<img>`로 교체 |
| 원장 사진 | 원장 소개 섹션 | placeholder를 `<img src="images/teacher.jpg">`로 교체 |

## 배포 (무료)
- **GitHub Pages**: 저장소에 push 후 Settings → Pages에서 활성화
- **Netlify / Vercel**: 폴더를 드래그&드롭하면 바로 배포

## 커스터마이징
색상은 `css/style.css` 맨 위 `:root` 변수만 바꾸면 전체에 반영됩니다.
